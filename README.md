# mikSQL
Experiments with user friendly MySQL administration tool.

В общем, мои попытки создать вменяемую утилиту для администрирования MySQL баз данных,
чтобы работало не только в винде, но и на Linux!

### Требования к системе
Разработку веду в Lazarus 2.0.2 на ОС Debian Testing (в данный момент, основной упор на Linux) и Windows 10.
